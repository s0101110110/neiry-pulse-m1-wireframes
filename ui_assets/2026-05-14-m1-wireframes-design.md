# Wireframes M1 · Tech Demo — Design Spec
Дата: 2026-05-14 | Проект: Neiry Pulse | Milestone: M1 · 18 мая

---

## Контекст

M1 (Tech Demo) — техническое подтверждение пайплайна: браслет → мобайл → backend → дашборд. Wireframes нужны Кириллу как reference для реализации и Никите для ревью скоупа до 18 мая.

---

## Решения

| Параметр | Выбор |
|---|---|
| Fidelity | Hi-fi, тёмная тема |
| Структура | Отдельный HTML на каждый экран + `index.html` с обоими флоу |
| Мобайл | В рамке телефона (SVG-силуэт смартфона) |
| Навигация | Два потока рядом на index: мобайл слева, дашборд справа |

---

wireframes/m1/

| Шрифт | Применение |

## Экраны · Мобайл (4 штуки)

Все мобайл-экраны — в рамке телефона, 390×844px (iPhone 14), тёмная тема.

### Экран 1 · Регистрация (`mobile-registration.html`)
- Логотип Neiry Pulse вверху (Fraunces)
- Заголовок «Создать аккаунт»
- Поле email
- Поле пароль
- Кнопка «Создать аккаунт» (accent `#831843`)
- Ссылка «Уже есть аккаунт? Войти» → `mobile-login.html`

### Экран 2 · Логин (`mobile-login.html`)
- Логотип Neiry Pulse вверху (Fraunces)
- Заголовок «Добро пожаловать»
- Поле email
- Поле пароль
- Кнопка «Войти» (accent)
- Ссылка «Нет аккаунта? Зарегистрироваться» → `mobile-registration.html`
- После логина → `mobile-ble-pairing.html`

### Экран 3 · BLE Pairing (`mobile-ble-pairing.html`)

Показывается **scanning state** — прямо в прототипе, без переключения:
- Заголовок «Подключить браслет»
- Пульсирующий круг-индикатор с BLE-иконкой в центре (CSS-анимация `pulse`, разрешена green-зоной)
- Подпись под кругом: «Поиск устройств…»
- Список найденных устройств (3 строки: `VITRO_A3F2`, `VITRO_B1C9`, `VITRO_D7E0`) — показываем сразу под индикатором
- У первой строки — кнопка «Подключить» (accent)
- После подключения → `mobile-home.html`

Пустое состояние не нужно для демо (всегда показываем 3 placeholder-устройства).

### Экран 4 · Главная (`mobile-home.html`)
- Статус браслета: зелёная точка `●` (`#166534`) + «Connected · VITRO_A3F2» (JetBrains Mono)
- BPM крупно: число `78` — JetBrains Mono, 72px, цвет `#831843`
- Индикатор тренда рядом с числом: стрелка `↑` / `↓` / `→` (вторичный текст)
- Подпись «уд/мин» под числом
- Время последнего замера (JetBrains Mono, вторичный цвет)
- Кнопка «Синхронизировать» (accent)
- Статус последней синхронизации

## Экраны · Dashboard (2 штуки)

Dashboard-экраны — полноширинные, без рамки телефона.

### Экран 5 · Логин администратора (`dashboard-login.html`)
- Логотип Neiry Pulse (Fraunces) + подпись «Admin Dashboard»
- Поле email
- Поле пароль
- Кнопка «Войти как администратор» (accent `#1e3a8a`)
- После логина → `dashboard-users.html`

### Экран 6 · Список пользователей (`dashboard-users.html`)
- Шапка: логотип + «Neiry Pulse · Admin» + кнопка «Выйти»
- Заголовок «Тестировщики» + счётчик `(4)`
- Таблица:
  - Имя пользователя
  - Email
  - Последняя синхронизация (JetBrains Mono)
  - BPM (JetBrains Mono, accent-цвет)
  - Статус — badge:
    - **Online**: `bg-[#166534]` + белый текст «Online»
    - **Offline**: `bg-[#2a2620]` + `text-[#7a766d]` «Offline»

## Index (`index.html`)

Два флоу рядом — мобайл слева, дашборд справа.

- Шапка: «Neiry Pulse · M1 Tech Demo Wireframes · 18 мая 2026»
- Разделитель с подписями **«Android App»** (слева) и **«Web Dashboard»** (справа)
- **Левая колонка** (мобайл): 4 телефонных рамки сверху вниз
  - Над каждой рамкой — номер шага `①` `②` `③` `④` + название экрана
  - Между рамками — стрелка `↓`
  - Под каждой рамкой — ссылка «Открыть полный экран →»
- **Правая колонка** (дашборд): 2 превью сверху вниз
  - Над каждым — номер `⑤` `⑥` + название
  - Ссылки на полные экраны
