Пробник моноширинных шрифтов

Критерии: ноль плоский (без точки/слэша) · 4 открытая сверху · 5 / 7 геометричные · кириллица

Подсвечены критичные глифы 0 4 5 7. Тоггл «Слэш в нуле» включает стандартную OpenType-фичу zero (диагональ) — где её нет, ноль не изменится. «Плоский без отметки» — это состояние при выключенном слэше: у Martian Mono / Spline Sans Mono он плоский нативно, у JetBrains/Plex/Source Code Pro в дефолте может оставаться точка — её убирает уже стилистический сет конкретного шрифта (номер сверь в Wakamai Fondue). Строка O0 О0 — проверка отличимости латинской/кириллической «О» от нуля.
