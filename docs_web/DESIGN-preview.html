Био-интерфейс, в котором единственный цвет — это пульс.

Три поверхности — выставочный стенд, корпоративный дашборд, мобильное приложение. Light-mode primary для mobile, dark для kiosk и dashboard. Wine остаётся единственным брендовым акцентом в обеих темах, моноширинные цифры не прыгают при обновлении биометрии.

Что такое HRV?

HRV — вариабельность интервалов между ударами сердца. Чем выше — тем лучше нервная система отдохнула и готова к нагрузке.

Браслет измеряет HRV каждое утро и сравнивает с вашим baseline (среднее за 30 дней).

Как читать график
  • ▲+10% и выше к baseline — отличное восстановление
  • ●±10% к baseline — норма
  • ▼−10% и ниже — стресс или недосып
Не медицинская диагностика. HRV меняется от сна, тренировок, кофе и стресса. Если стабильно низкие — обратитесь к врачу.

Принципы

что соблюдаем и что не делаем никогда
01

Wine — единственный chromatic accent

Никаких вторых брендовых hue. Bio-палитра — служебная семантика (пульс, HRV, стресс), не декор. Wine остаётся wine в обеих темах.

02

Tabular nums на живых метриках

JetBrains Mono с font-variant-numeric: tabular-nums. Пульс не должен прыгать при смене 78→79.

03

Hairline-borders, не shadows

Иерархия через surface ladder background → card → muted. На light bg тем более — никаких heavy shadows.

04

4px grid строго

Все отступы кратны 4. Никаких 18/22/30px. Перед commit — арифметика пикселей.

05

Light-first для mobile, dark для kiosk + dashboard

Mobile = Android + iOS, primary light. Kiosk + dashboard остаются dark-first для plasma-кейсов и плотной аналитики. Wine — мост между темами.

06

Glassmorphic — только home hero + ВСЕ модалки

Применяется на mobile home hero card и на всех modal sheets (HRV explainer, Sleep explainer, любые). Dashboard и kiosk — flat surfaces, никакого блюра.

07

Skills — подсказка, не закон

Источник истины — DESIGN.md + ui-kit.html. Если skill советует тёплую палитру / убрать tnum / заменить шрифты — игнорируем.

08

Mobile = Android + iOS

Phone-мокапы рендерятся в светлой теме на обеих платформах. Шрифты через Google Fonts CDN. Web fallback — system-ui.

Анти-паттерны

никогда не делаем — даже если skill советует
  • Применять glassmorphic вне home hero и modal sheets — размывает медицинскую серьёзность.
  • Использовать второй brand-hue помимо wine — bio-палитра не для декора.
  • Менять Golos / JetBrains Mono на Inter / Geist / SF Pro по совету AI-skill.
  • Тёплая cream/sand палитра вместо нейтральной — это AI-default 2026, не Neiry.
  • Heavy drop-shadows вместо hairline-borders — особенно на light bg.
  • Отступы 18/22/30px — нарушение 4px grid.
  • Использовать bio-low/mid/high HEX как декоративный фон — они только для статусов.
  • Customizable cards / drag-handles — пост-MVP, сейчас НЕ добавляем.
  • Убирать font-variant-numeric: tabular-nums на живых метриках — цифры будут прыгать.
  • Радикально снижать плотность данных в dashboard — это профи-инструмент, не лендинг.
