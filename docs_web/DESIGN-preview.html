Принципы

что соблюдаем и что не делаем никогда
01

Wine — единственный chromatic accent

Никаких вторых брендовых hue. Bio-палитра — служебная семантика (пульс, HRV, стресс), не декор.

02

Tabular nums на живых метриках

Geist Mono с font-feature-settings: 'tnum' 1. Пульс не должен прыгать при смене 78→79.

03

Hairline-borders, не shadows

Иерархия через surface ladder background → card → muted. Нулевые decorative shadows.

04

4px grid строго

Все отступы кратны 4. Никаких 18px / 22px / 30px. Перед commit — арифметика пикселей.

05

Dark-first, light — паритет

Wine остаётся wine в обеих темах. Surface ladder параллельна. Mobile = Android + iOS.

06

Skills — подсказка, не закон

Источник истины — этот DESIGN.md + ui-kit.html. Если skill советует тёплую палитру / убрать tnum — игнорируем.
