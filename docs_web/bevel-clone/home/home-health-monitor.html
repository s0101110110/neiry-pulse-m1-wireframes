Возражения skill'ов — на разбор PM

  • impeccable typeset / 14px label: на 360-px телефоне label 14px + цифра 28px создают «короткий» вертикальный ритм внутри карточки. Можно опустить label до 13px, но тогда теряется правило DESIGN.md «card-title 18px / body 14px» → решено: оставить 14px, иерархия идёт через вес и цвет (label = ink-mute medium 500, цифра = ink mono 500).
  • redesign-existing-projects / capsule fill direction: в Bevel-референсе fill идёт от низа к dot — это семантически означает «доля от минимума личной нормы». Альтернатива: fill от центра в обе стороны (как градиент severity). Решено: оставить Bevel-логику (fill bottom→dot) для строгого соответствия референсу и предсказуемой логики разработчику.
  • typeset / Pulse Mono CDN: локальный @font-face Pulse Mono из v2 здесь не доступен (cwd bevel-clone/home/). Использован стек 'Pulse Mono', 'JetBrains Mono' (Google Fonts), ui-monospace — JetBrains Mono это исходник Pulse Mono (DESIGN.md), визуально идентичен. Если PM хочет именно локальный Pulse Mono — нужен относительный путь ../../../UI_assets/fonts/pulse-mono/.
  • colorize-rules / Темп = red: 38.1°C — фебрильная зона, формально «Critical». В тексте оставлено «Внимание» (русский UI без англицизмов, memory feedback_neiry_ui_no_anglicisms). Цвет красный — корректно по severity ladder Bevel.
