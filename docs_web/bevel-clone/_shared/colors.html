3 · FAIL notes

5 пар не дотягивают до AA-large (≥3). Все — комбинации ink-dim (#8E8E97) на разных фонах. ink-dim никогда не должен нести смысловой текст — только декоративные lookups / иконки.

  • FAIL
    ink-dim · bg-screen 2.86:1
    Не использовать для текста. Допустимо только для tab-label иконок (декоративные стрелки, точки-маркеры) и hairline-разделителей под крупным заголовком, где значение дублируется body-копией.
  • FAIL
    ink-dim · wine-tint 2.65:1
    Самая низкая пара. Запрет на текст. Wine-tint используется только для активной nav-кнопки — в ней текст идёт var(--bv-wine), не ink-dim.
  • FAIL
    ink-dim · sem-norm-bg 2.92:1
    В green chip-ах текст всегда sem-norm-ink (4.88:1 AA). ink-dim допустим лишь как icon-stroke ≥ 2px (декоративный чек-mark, не несёт текст).
  • FAIL
    ink-dim · sem-warn-bg 2.75:1
    В warn chip-ах текст — sem-warn-ink (3.96 AA-LG, ≥18pt) или ink (14.50 AAA для body). ink-dim — только icon-decorative.
  • FAIL
    ink-dim · sem-high-bg 2.56:1
    Critical chip требует высокого контраста для readability. Текст: sem-high-ink (4.88 AA) или ink (13.51 AAA). ink-dim — запрет.
