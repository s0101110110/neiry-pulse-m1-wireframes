Симуляция флоу приложения ★

Экраны соединены стрелками как в реальном приложении: Регистрация → BLE Pairing → Главная. На каждом экране — кнопки «Далее» и «Назад» которые переходят к следующему логическому шагу. Index показывает всю схему с превью.
