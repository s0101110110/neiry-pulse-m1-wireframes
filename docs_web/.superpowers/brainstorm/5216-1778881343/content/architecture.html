🏁 milestones/ — M1, M2, M3: скоуп, статус, дедлайн
⚡ decisions/ — зафиксированные решения из встреч
❓ open-questions/ — нерешённые вопросы с контекстом
👥 people/ — участники, роли, зоны ответственности

⚙️ Как работает агент

1. Старт сессии
Читает PROJECT_STATE.md
Проверяет INBOX/
Знает контекст сразу
2. Обработка
Транскрипция → tasks/
Документ → decisions/
Таблица → milestones/
3. Конец сессии
Обновляет PROJECT_STATE.md
Коммитит изменения
INBOX/ очищен
