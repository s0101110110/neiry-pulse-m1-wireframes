Neiry Pulse · UI Kit

Живой стайлгайд дизайн-системы. Семантические токены shadcn (zinc base) + Neiry wine accent (#831843), две темы (тёмная для плазмы, светлая для админ-ноута), Manrope/Fraunces/JetBrains Mono. Все компоненты для рестайла собраны в одном месте: токены, base shadcn-компоненты, кастомные био-виджеты и Tremor-стиль для дашборда корпоратов.

01

Tokens

Семантические переменные shadcn — все компоненты ссылаются только на них. При смене темы меняется CSS-переменная, а не код компонента.

Когда применять

--primary — CTA, акценты, активные табы, фокус-ринги
--chart-2 — success / норма BPM / Recovery OK
--chart-4 — warning / каскад / повышенный stress

--muted-foreground — подписи, secondary-текст, иконки

3.6 Heartbeat pulse animation

CSS-анимация для live-BPM. Скорость синхронизируется с реальным BPM (700мс ≈ 86 BPM). Иконка масштабируется 1.0 → 1.15 → 1.0.

animate-heart · 750ms

Применение: декоративная индикация активного соединения с браслетом на kiosk и в мобильном приложении.

@keyframes heart-pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.15); }
}
.animate-heart { animation: heart-pulse 0.75s ease-in-out infinite; }
